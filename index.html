JavaScript

JavaScript — это «безопасный» язык программирования. JavaScript на веб-странице не может:

  • читать/записывать произвольные файлы на жёстком диске
  • копировать файлы на диске
  • запускать программы на диске
  • не имеет прямого доступа к системным функциям ОС

JavaScript

Страница с поддержкой JavaScript не может незаметно, например, включить веб-камеру/микрофон, наблюдать за происходящим и отправлять информацию. Эти способы взаимодействия требуют явного разрешения пользователя

Same Origin Policy - Политика одинакового источника

JavaScript с одной страницы не имеет доступа к другой, если переход происходит с разных сайтов (с другого домена, протокола или порта).

Разные окна/вкладки не знают друг о друге

JavaScript

JavaScript – это браузерная технология, которая:

  1. имеет полную интеграцию с HTML/CSS
  2. поддерживается всеми основными браузерами и включёна по умолчанию

Брендан Айк

Брендан Эйх или Айк (англ. Brendan Eich, родился 4 июля 1961, Питтсбург, Пенсильвания, США) — американский программист, создатель языка программирования JavaScript. С 1995 года работал в компании Netscape; участвовал в основании Mozilla, до 2014 года был техническим директором в Mozilla Corporation. В марте 2014 года был назначен CEO, но вскоре покинул Mozilla под давлением недовольных его назначением из-за поддержки им в 2008 году запрета однополых браков. На данный момент работает исполнительным директором Brave Software wikipedia .

ECMAScript

ECMA — ассоциация, созданная в 1961 году с целью стандартизации информационных и коммуникационных систем

На сегодняшний день JavaScript это всего лишь коммерческое название ECMAScript

ECMAScript 3

ECMAScript 3 был выпущен в декабре 1999 года c первыми большими изменениями языка

AJAX

AJAX (asynchronous JavaScript and XML) — технология, которая позволяла браузеру выполнять асинхронные HTTP-запросы серверу, тем самым позволяя страницам обновляться на лету

AJAX позже было стандартизировано группами WHATWG и W3C

Asm.js — это небольшое, более строгое подмножество JavaScript которое ограничивает стандартный язык только конструкциями, типа while, if и данными в виде чисел, именованных функций, и другими простыми вещами

Вывод в консоль

Для вывода данных в консоль обратитесь к объекту console и его методу log:

      console.log("Какая-либо информация");

Переменные

  1. var — это устаревший способ объявления
  2. let — это современный способ объявления
  3. const — похоже на let, но значение переменной не может изменяться
      var a = "Это строка";
      let b = 42;
      const c = {
        name: "Alex"
      };

Переменные

  1. имя переменной должно содержать только буквы, цифры или символы $ и _
  2. первый символ не должен быть цифрой
      let user;
      const user1;
