JavaScript — это «безопасный» язык программирования. JavaScript на веб-странице не может:

  • читать/записывать произвольные файлы на жёстком диске
  • копировать файлы на диске
  • запускать программы на диске
  • не имеет прямого доступа к системным функциям ОС

Условное ветвление

Иногда нам нужно выполнить различные действия в зависимости от условий

      if (year == '2024') {
        console.log("Это текущий год");
      }

Преобразование к логическому типу

  1. Число 0, пустая строка "", null, undefined и NaN становятся false. Из-за этого их называют «ложными» («falsy») значениями
  2. Остальные значения становятся true, поэтому их называют «правдивыми» («truthy»)
      if ("Строка") { // "Строка" преобразовывается в true
        console.log("Это правдивое значение");
      }

Блок «else»

      if (year === '2024') {
        console.log("Это верно");
      } else {
        console.log("Это не верно");
      }

Блок «else if»

      if (age < 18) {
        console.log('Потребовать паспорт');
      } else if (age >= 18 && age < 30) {
        console.log('Потребовать паспорт (на всякий случай)');
      } else {
        console.log('Предложить скидку');
      }

Условный (тернарный) оператор

Прерывание цикла

Обычно цикл завершается при вычислении условия в false. Но мы можем выйти из цикла в любой момент с помощью специальной директивы break 

      for (let i = 0; i < 10; i++) {
        if (i == 5) {
          break;
        }
        // что-то делаем
      }

Переход к следующей итерации

При выполнении директивы continue цикл не прерывается, а переходит к следующей итерации

      for (let i = 0; i < 10; i++) {
        if (i % 2 == 0) continue;
        // что-то делаем
      }

Массивы

Массив (объект Array) — структура данных, которая позволяет хранить коллекцию из нескольких элементов

      const cats = ['Murzik', 'Barsik'];
      const nums = [42, 1, -5, 0];
      const some = [81, 'message', true, { a: 1, b: 2}, [0, 1]];

Создание массивов 

      let arr1 = new Array();
      let arr2 = ['white', 'blue', 'red'];

Чтобы получить элемент, указав его номер в квадратных скобках

      console.log(arr2[0])
      console.log(arr2[1])
      console.log(arr2[2])

Методы массивов

arr.[название метода]()

  1. pop — удаляет последний элемент из массива
  2. push — добавляет элемент в конец массива
  3. shift — удаляет из массива первый элемент
  4. unshift — добавляет элемент в начало массива

Перебор элементов массива

Для массивов возможен также перебор с помощью цикла for..of

      for (let color of colors) {
        console.log( color );
      }

Цикл for..of не предоставляет доступа к номеру текущего элемента, только к его значению

Объекты

Объекты – это ассоциативные массивы с рядом дополнительных возможностей. Они хранят свойства (пары ключ-значение), где:

  • ключи свойств должны быть строками или символами
  • значения могут быть любого типа
      const messages = {
        'msg': 'Ошибка доступа' ,
        'code': 0
      }

Объекты

Чтобы получить доступ к свойству, мы можем использовать:

  1. obj.property — через точку
  2. obj["property"] — с помощью квадратных скобок
      console.log(messages.code) // "0"
      console.log(messages['msg']) // "Ошибка доступа"

Перебор свойств объекта

Для объектов возможен перебор с помощью цикла for..in

      for (let item in colors) {
        console.log( colors[item] );
      }

Создание объектов

Методы для работы со строками

Методы toLowerCase() и toUpperCase() меняют регистр символов:

      console.log( 'Text'.toUpperCase() ); // TEXT
      console.log( 'Text'.toLowerCase() ); // text

Методы для работы со строками

Для поиска подстроки используйте indexOf или includes/startsWith/endsWith, когда надо только проверить, есть ли вхождение

      let str = 'Some text';
      console.log(str.indexOf('Some')); // 0 
      console.log(str.includes('Some')); // true

Получение подстроки

substring, substr и slice

  1. slice(start, end): ищет от start до end (не включая end)
  2. substring(start, end): ищет между start и end (не включая end)
  3. substr(start, length): ищет length символов, начиная от start

Сравнение строк

Строки сравниваются посимвольно в алфавитном порядке, но:

  1. строчные буквы больше заглавных
  2. буквы, имеющие диакритические знаки, могут идти «не по порядку»
      console.log('z' > 'a') // false
      console.log('a' > 'Z') // true
      console.log('Österreich' > 'Zealand') // true

Правильное сравнение строк

все современные браузеры поддерживают стандарт ECMA 402, обеспечивающий правильное сравнение строк на разных языках с учётом их правил

      str.localeCompare(str2) // вернет число
  • 0, если строки равны
  • положительное число, если str больше str2
  • отрицательное число, если str меньше str2
